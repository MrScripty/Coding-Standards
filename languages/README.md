# Language-Specific Standards

Language-specific standards extend the generic standards where a language or
toolchain provides stronger conventions, verification tools, or architectural
constraints.

## Languages

| Language | Standards |
| --- | --- |
| Rust | [Rust Standards](rust/RUST-STANDARDS.md) |

Language-specific details should not live inline in the root standards. Keep the
root standards focused on cross-language principles, then link to the matching
language standard for toolchain-specific rules and examples.
